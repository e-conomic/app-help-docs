Shortcuts - Top Left Button area

ctrl + shft + m Open 'Topleft Drop Down with journals and lists' drop-down (choose option in drop-down by selecting number)
ctrl + shft + n Create new line in current journal (This shortcut does not work in Google Chrome Browser)
ctrl + shft + d Delete selected entries or current entry
ctrl + shft + o Open account, supplier or customer lists
ctrl + shft + t Show transactions on Ledger Card
ctrl + shft + 1 Book selected entries or current entry
ctrl + shft + s Save current entry

List view & navigation Shortcuts
esc Undo cell editing
ctrl + ent Change to Form View
↑/↓ or ←/→ Move up/down or left/right
ctrl + ↑/↓ Move between sections
shft + ↓ Create new entry below current cell
home or end Move to first or last cell in current entry
ctrl + home Move to Top cell in current column
ctrl + end Move to Buttom cell in current column
tab or ent Move to next cell in current entry
shft + tab Move to previous cell in current entry
F12 Edit cell
spc Select/Deselect checkbox in current entry
ctrl + shft + a Select/deselect all checkboxes
ctrl + shft + f Move til filter cell in current column
ctrl + shft + z Copy Data from cell above

Document Handling Shortcuts - in Journal
shft + spc Show Document in Full page
shft + spc + +/- Zoom in/out on Document
shft + spc + ↑/↓ Scroll up/down on Document page
shft + space + ←/→ next/previous page
