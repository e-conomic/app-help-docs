Shortcuts - Top Left Button area

ctrl + shft + m Open 'Topleft Drop Down with journals and lists' drop-down - (choose option in drop-down by selecting number)
ctrl + shft + d Delete selected entries or current entry
ctrl + shft + o Open account, supplier or customer lists
ctrl + shft + t Show transactions on Ledger Card
ctrl + shft + 1 Book selected entries or current entry
ctrl + shft + s Save current entry

List view & navigation Shortcuts
esc + Undo cell editing
ctrl + ent + Change to Form View
ctrl + shft + a Select/Deselect all lines
ctrl + shft + a Select/Deselect all lines
ctrl + shft + z Copy Data from cell above
shft + arrow down Create new entry below current
Ent or Tab Move to next cell in current row
shft + Ent Move to previous cell in current row

Document Handling Shortcuts
shft + spc + Show Document in Full page
shft + spc + +/- Zoom in/out on Document
shft + spc + ↑/↓ Scroll up/down on Document page
shift + space + ←/→ next/previous page
